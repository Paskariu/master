# Bruteforce

Idee: Alles ausprobieren
Beispiel: Finde zwei Zahlen mit Summe = x -> alle Paare Prüfen
Wann
+ n kleiner/gleich 20-25
+ "alle Möglichkeiten"
+ keine bessere Struktur erkennbar

# Sortieren
Idee: Erst sortieren, dann einfach lösen
Beispiel: Finde zwei Zahlen mit Summe x -> Sortieren + two pointers
Wann:
+ Reihenfolge egal
+ "kleinstes/größtes/nächstes"
+ Vergleiche zwischen Elementen
Komplexität meistens: O(n log n)
